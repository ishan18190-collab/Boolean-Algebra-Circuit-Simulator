# Boolean Algebra Circuit Simulator

## Overview
This repository contains the source code for a Boolean Algebra Circuit Simulator, developed as a Discrete Mathematics project for BML Munjal University. The simulator accepts user-defined Boolean expressions, maps them to physical logic gates, and applies mathematical laws to minimize the circuit complexity.

## Features
* **Expression Parsing:** Converts text-based Boolean expressions into logic circuits.
* **Supported Logic Gates:** AND, OR, NOT, NAND, NOR, XOR.
* **Truth Table Generation:** Automatically calculates exhaustive truth tables for given inputs.
* **Circuit Minimization:** Applies Boolean identities and minimization algorithms to optimize gate count.
